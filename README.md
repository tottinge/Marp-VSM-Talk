# VSM Marp Socrates

Image-forward Marp slideshow scaffold with sparse text defaults.

## Requirements
- Node.js 20+

## Setup
```bash
npm install
```

## Development
```bash
npm run preview
```

## Build exports
```bash
npm run build
```

Outputs are written to `output/` as HTML, PDF, and PPTX.

## Test/validation
```bash
./run_tests
```
