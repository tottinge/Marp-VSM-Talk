# VSM Marp Socrates

Image-forward Marp slideshow scaffold with sparse text defaults.
## Workshop source docs

- `Faster-and-More-Predictable-Workshop-Summary.md` (primary workshop design)
- `Workshop-Principles-Sources.md` (authoritative principle lineage and source mapping)
- `roots.md` (supporting bibliography and conceptual roots)
- `Source-Type-Whyitmatters.csv` (canonical source/type/why-it-matters matrix)
- `AGENTS.md` (operational guidance for future AI-assisted edits)

## Requirements

- Node.js 20+

## Setup

```bash
npm install
```

## Development

```bash
npm run preview
```

## Build exports

```bash
npm run build
```

Outputs are written to `output/` as HTML, PDF, and PPTX.

## Test/validation

```bash
./run_tests
```
